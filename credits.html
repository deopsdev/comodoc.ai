Sources & Licenses

This project links to and may call inference services and open‑source models. Below are the main sources and brief license notes — check each provider for full terms.

Models & Providers

  • LLaMA (Meta) — model family from Meta AI. See official page: ai.meta.com/llama. Check the model license and usage restrictions per release.
  • Hugging Face — model hosting & inference API. API keys & quotas required for hosted inference. Docs: huggingface.co/docs.
  • Pollinations.ai — used as a fallback inference endpoint in this project. Availability may vary (Cloudflare tunnel used by provider).

Privacy & Data

The application is a client for third‑party inference APIs. Do not send sensitive PII or secrets to the service unless you understand the provider's data handling and retention policy.

Licence pointers

  • Open‑source models may have specific licenses (MIT, Apache‑2.0, custom). Always review the license for the exact model you use.
  • Hosted provider APIs (Hugging Face, Pollinations, etc.) have separate Terms of Service — check for commercial restrictions.

Where to get API keys

  • Hugging Face: create an account and get an access token at huggingface.co/settings/tokens.
  • Other providers: follow their docs (links above).
